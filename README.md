# axiom-pro
AXIOM PRO v4.0 — Multiphysics Engineering Simulation Suite  A fully interactive browser-based FEA simulation environment built with  Three.js. Features a live 3D viewport, parametric model editor, engineering  problem solver (Machine Design, Fluid Mechanics, Heat Transfer, Structural  Analysis, Thermodynamics, Dynamics), Built by ENG_ALAA MOHAMMED.
